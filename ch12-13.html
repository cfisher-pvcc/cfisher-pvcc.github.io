Chapter 12: Models of Computation
  1. What is the purpose of a model? A model's purpose is to capture the essence (important properties) of the real thing on a much smaller scale, omits some details of the real thing and lacks full functionality.
List four things a computing agent should be able to do:
  1. Accept input
  2. Store information in memory and retrieve it from memory
  3. Take actions according to algorithm instructions; the choice of what action to take may depend on the present state of the computing agent, as well as the input item presently being processed.
  4. Produce Output
List four characteristics if a algorithm:
  1. Be a well-ordered collection
  2. Consist of unambiguous and effectively computable operations
  3. Halt in a finite amount of time
  4. Produce a result
What is the "halting problem" with regards to an uncomputable problem?
  1. The halting problem is the problem of determining, from a description of an arbitrary computer program and an input, whether the program will finish running, or continue to run forever.
Chapter 13: Simulation and Modeling
Define These Terms:
  1. Scientific Method:Entails observing the behavior of a system and formulating a hypothesis that tries to understand and explain that behavior.
  2. Computational Model:A physical system is not modeled as a small scale replica but as a set of mathematical equations and or algorithmic procedures that capture the fundamental properties and behaviors of a system.
  3. Discrete Event Simulation:a method used to model real world systems that can be decomposed into a set of logically separate processes that autonomously progress through time.
  4. List and Describe Seven Reasons to Construct a Simulation Model:
    • Existence: The system might not yet exist, so it is not possible to experiment directly on the actual system.
    • Physical Realization: The system is not constructed from entities that can be represented by physical objects.
      • Safety: It might be to dangerous to experiment on the actualy system or physical replica.
    • Speed of Construction: It might take to much time to construct a physical model.
    • Time Scale: Some physical systems change too slowly or too quickly.
    • Ethical Behavior: Some physical models have serious moral and ethical consequences.
    • Ease of Modification: If we are not happy with the design of a physical model, we need to construct a brand new one. in a simulation, we only need to change some numerical parameters and rerun the existing model.
Illustris: "The Mother of All Computation":
  1. What is Illustris? The Illustris project is an ongoing series of astrophysical simulations run by an international collaboration of scientists. The aim was to study the processes of galaxy formation and evolution in the universe with a comprehensive physical model.
  2. What fields of science were involved in the Illustris compuational model?: It covers multiple disciplines such as physics, astronomy, and computer science.
  3. What does Illustris simulate?: Galaxy formation
  4. How long did the project take?: 5 years
  5. How many years of galactic evolution are simulated in Illustris?: 13.8 billion years
  6. Illustris was created using two supercomputers. what is a supercomputer?A supercomputer is a computer with a high level of performance as compared to a general-purpose computer. The performance of a supercomputer is commonly measured in floating-point operations per second instead of million instructions per second.
